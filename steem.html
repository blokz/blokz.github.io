Tool : Check Steem Voting Power

Instructions:

  • Update "sn0n" in the text field below to your name
  • press the Green "Run" button
The status will update to 'running' in green next to 'help' while processing, and will update after a few moments.

var steem = require('steem'); steem.config.set('websocket','wss://rpc.buildteam.io'); var accounts = steem.api.getAccounts(["sn0n"], function(err, response){ var secondsago = (new Date - new Date(response[0].last_vote_time + "Z")) / 1000; var vpow = response[0].voting_power + (10000 * secondsago / 432000); vpow = Math.min(vpow / 100, 100).toFixed(2); console.log("Voting Power: " +vpow); }); // notes : https://www.steem.center/index.php?title=Public_Websocket_Servers
